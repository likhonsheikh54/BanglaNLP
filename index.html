BanglaNLP

A comprehensive open-source toolkit for Bengali natural language processing, featuring tokenization, named entity recognition, sentiment analysis, and text classification capabilities.

Get Started

Features

T

Tokenization

Advanced tokenization specifically designed for Bengali language structure, supporting word, sentence, and subword tokenization.

N

Named Entity Recognition

Identify and classify named entities in Bengali text, including people, organizations, locations, and more.

S

Sentiment Analysis

Analyze the sentiment of Bengali text to determine if the expressed opinion is positive, negative, or neutral.

C

Text Classification

Classify Bengali text into predefined categories with high accuracy using state-of-the-art machine learning models.

D

Datasets

Access to curated Bengali language datasets for training and testing your NLP models.

P

Pre-trained Models

Use our pre-trained models to quickly implement NLP capabilities in your Bengali language applications.

Code Example

# Install BanglaNLP
pip install bangla-nlp

# Import the library
from bangla_nlp import BanglaNLP

# Initialize with the desired components
nlp = BanglaNLP(components=["tokenizer", "ner", "sentiment"])

# Process text
text = "আমি বাংলায় প্রাকৃতিক ভাষা প্রক্রিয়াকরণ করছি।"
doc = nlp(text)

# Access tokens
for token in doc.tokens:
  print(token.text, token.pos)

# Get named entities
for entity in doc.entities:
  print(entity.text, entity.label)

# Get sentiment
print(f"Sentiment: {doc.sentiment}")
